Rule based G2P / micro service API

List available languages

G2Ps: /g2p/list
Syllabifiers: /syll/list

Display rule content

URL: /rules/LANG

Examples:
/g2p/rules/sws
/g2p/rules/basque_ipa

Transcribe a word (JSON)

Transcribe an input word using the specified language's g2p rules and return the result as json

URL: /transcribe/LANG/WORD

Switches
format: xml, txt, json (default)
Examples:
/transcribe/ipa_test/hot
/transcribe/sws/hit
/transcribe/sws/hit?format=xml
/transcribe/sws/hit?format=text
/transcribe/sws/dusch

Transcribe a word (XML)

Transcribe an input word using the specified language's g2p rules and return the result as xml

URLS: /transcribe/LANG/WORD?format=xml, /xmltranscribe/LANG/WORD

Examples:
/transcribe/ipa_test/hot?format=xml
/transcribe/sws/hit?format=xml
/transcribe/sws/dusch?format=xml

For legacy calls from ltool/yalt:
/xmltranscribe/ipa_test/hot
/xmltranscribe/sws/hit
/xmltranscribe/sws/dusch

Divide a transcription into syllables

Divide a transcription into syllables based on the specified language's syllable definition (if any)

URL: /syllabify/LANG/TRANS

Examples:
/syllabify/sws/d%20u0%20S%20a"
