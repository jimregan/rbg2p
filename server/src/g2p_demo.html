Rule based G2P / micro service API

List available languages

URL: /rbg2p/list

Transcribe a word

Transcribe an input word using the specified language's g2p rules

URL: /rbg2p/transcribe/LANG/WORD


Examples:
/rbg2p/transcribe/ipa_test/hot
/rbg2p/transcribe/sws_test/hit
/rbg2p/transcribe/sws_test/dusch
