Rule based G2P / micro service API

List available languages

URL: /rbg2p/list
Params: none

Transcribe one or more words

Transcribe an input word using the specified language's g2p rules

URL: /rbg2p/transcribe
Params: lang, words (required)

Examples:
/rbg2p/transcribe?lang=ipa_test&words=hit
/rbg2p/transcribe?lang=sws_test&words=hit
